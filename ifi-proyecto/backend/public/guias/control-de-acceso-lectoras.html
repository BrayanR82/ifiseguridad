Configuración de lectoras de control de acceso

Guía para integrar lectoras y controladoras con criterios de seguridad física, continuidad operativa y control granular por perfiles.

Aplica tanto para accesos peatonales como para áreas restringidas donde se requiere bitácora detallada de entradas, salidas y eventos de excepción.

Antes de empezar

  • Controladora compatible, lectoras correctamente cableadas y fuente estabilizada.
  • Definición previa de niveles de acceso, horarios y excepciones operativas.
  • Catálogo de usuarios o credenciales iniciales para pruebas de aceptación.

Paso a paso

  1. Revisa cableado Wiegand/OSDP, alimentación y tierra antes de energizar.
  2. Registra la lectora en la controladora y asigna puerta o zona correspondiente.
  3. Configura horarios, niveles de acceso y reglas anti-passback si aplica.
  4. Da de alta tarjetas o credenciales y asocia usuarios con su perfil.
  5. Valida aperturas, rechazos, eventos de puerta forzada y bloqueo remoto.
  6. Respalda la configuración y deja evidencia de pruebas de aceptación.

Buenas prácticas

  • Evita compartir credenciales entre áreas para mantener auditoría real por usuario.
  • Implementa prueba de contingencia ante corte de energía y verifica modo fail-safe/fail-secure.
  • Respalda base de datos de credenciales antes de cambios masivos de configuración.
