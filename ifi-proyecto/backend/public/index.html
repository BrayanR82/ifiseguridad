Preguntas Frecuentes

Resolvemos tus dudas sobre nuestra plataforma

¿Es compatible con mis cámaras actuales? +
Sí, IFI Seguridad es compatible con la mayoría de cámaras IP del mercado. Nuestro equipo técnico puede evaluar tu sistema actual sin costo y determinar la mejor forma de integración. Además, ofrecemos adaptadores y conversores para modelos más antiguos.
¿Cómo funciona el almacenamiento en la nube? +
Todas las grabaciones se envían automáticamente a servidores seguros con encriptación de extremo a extremo. Puedes acceder a ellas desde cualquier dispositivo durante el período de retención de tu plan (30 días para Pro, ilimitado para Enterprise). La transferencia es automática y no requiere configuración.
¿Tienen servicio de respuesta física? +
Sí, contamos con convenios con empresas de seguridad física en las principales ciudades. Al detectarse una alarma crítica, nuestro centro de monitoreo puede coordinar el envío de personal especializado a tu ubicación en cuestión de minutos. Este servicio se contrata por separado.
¿Qué pasa si se corta internet? +
El sistema continúa grabando localmente en el almacenamiento interno de las cámaras. Cuando la conexión se restablece, todo se sincroniza automáticamente a la nube. Recibirás una notificación si se detecta una desconexión prolongada.
¿Cuánto cuesta la instalación? +
Ofrecemos instalación profesional desde $799. El costo varía según la cantidad de cámaras, cableado requerido y complejidad del proyecto. Realizamos una consulta gratuita para proporcionarte un presupuesto exacto.
